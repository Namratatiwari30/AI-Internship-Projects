# Artificial Intelligence Internship - Completed Projects

This repository contains my required project submissions for the successful completion of the Artificial Intelligence Internship program.

---

## 📈 Project 1: Stock Price Predictor
A supervised machine learning model that forecasts historical and live daily stock market close values using a Linear Regression algorithm trained on real-time financial metrics.

* **Core Architecture:** Linear Regression via `scikit-learn`
* **Data Integration:** Live automated market data streams via `yfinance`
* **Model Accuracy achieved:** 99% (0.99 R² Evaluation Score)
* **Script File:** `predict.py`

---

## 🎬 Project 2: Movie Recommendation System
A content-based recommendation engine that vectorizes textual film parameters and calculates data similarities to suggest top matching alternative options.

* **Core Architecture:** Cosine Similarity & TF-IDF Vectorization via `scikit-learn`
* **Data Layout:** High-density multidimensional feature tagging array models
* **Script File:** `movie_recommend.py`
